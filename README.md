# snowdom
>try using snowpack and effector

## Get Started
### Installation
* Install node_modules
```
yarn install
```
* Prepare snowpack modules
```
yarn prepare
```
### Usage

Run `tarn start` to start server with live-reload and serve application.

Edit code and refresh page to see changes
